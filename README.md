# 📖 About the Internship

The **Celebal Excellence Internship (CEI)** is an industry-oriented internship program designed to bridge the gap between academic learning and real-world software development.

Throughout the internship, I worked on practical assignments and AI-driven projects covering **Machine Learning, Deep Learning, Computer Vision, Natural Language Processing, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and modern AI application development**.

This repository serves as a record of my internship journey, showcasing my progress, technical growth, and practical implementation of concepts learned during the program.
