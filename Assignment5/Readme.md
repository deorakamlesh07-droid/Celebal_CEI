# 📚 Text Generation Using Vanilla RNN, LSTM, and GRU

## 🚀 Overview

This project implements and compares three recurrent neural network architectures—Vanilla RNN, LSTM, and GRU—for text generation. The models learn language patterns, contextual dependencies, and word sequences from a custom text corpus and generate meaningful text through next-word prediction.

## 🎯 Objective

To design and evaluate deep learning models capable of learning the structure and grammar of a text corpus and generating coherent text sequences using:

* Vanilla RNN
* LSTM (Long Short-Term Memory)
* GRU (Gated Recurrent Unit)

## ✨ Features

* Custom text corpus preprocessing
* Text tokenization and vocabulary creation
* N-gram sequence generation
* Sequence padding using Keras
* Implementation of Vanilla RNN, LSTM, and GRU models
* Training with Adam optimizer
* Loss and accuracy visualization
* Text generation using next-word prediction
* Comparative analysis of model performance

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* TensorFlow / Keras

## ⚙️ Workflow

1. Text Cleaning and Preprocessing
2. Tokenization and Vocabulary Building
3. N-gram Sequence Generation
4. Data Preparation
5. Model Development (RNN, LSTM, GRU)
6. Model Training
7. Performance Visualization
8. Text Generation
9. Model Comparison and Analysis

## 📊 Results

* Compared Vanilla RNN, LSTM, and GRU using the same dataset and training pipeline.
* Visualized cross-entropy loss and accuracy trends.
* Generated text from a common seed phrase for all models.
* Observed that LSTM and GRU handled long-term dependencies more effectively than Vanilla RNN.

## 🎓 Learning Outcomes

* Understanding of sequence modeling and recurrent neural networks.
* Practical implementation of RNN, LSTM, and GRU architectures.
* Experience with text preprocessing and next-word prediction.
* Analysis of model performance for text generation tasks.
