# Customer Intelligence System using Classification, Ensemble Learning & Clustering

## Overview

This project develops an end-to-end Customer Intelligence System using machine learning techniques to identify meaningful customer segments and predict segment membership. The workflow combines clustering and ensemble learning approaches to generate actionable insights from the data.

## Features

* Data Preprocessing and Cleaning
* Exploratory Data Analysis (EDA)
* Feature Scaling using StandardScaler
* K-Means Clustering
* DBSCAN Clustering
* PCA-based Cluster Visualization
* Random Forest Classification
* XGBoost Classification
* Hyperparameter Tuning using RandomizedSearchCV
* Model Performance Evaluation
* Customer Segmentation Insights
* Overfitting and Underfitting Analysis

## Machine Learning Techniques

### Clustering

* K-Means
* DBSCAN

### Ensemble Learning

* Random Forest Classifier
* XGBoost Classifier

### Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Matplotlib
* Seaborn

## Results

The project successfully segments customers into meaningful groups using clustering techniques and predicts segment membership using ensemble learning models. Hyperparameter tuning and model evaluation were performed to achieve strong predictive performance while maintaining good generalization.
